List dalam HTML

Untuk membuat list dalam html, kita bisa menggunakan tag <ul> dan <li>. Berikut ini adalah contoh codingan nya

<!DOCTYPE html>
<html>
  <head>
    <title> Contoh Ordered List dalam Html </title>
  </head>
  <body>
    <p>Daftar Nama Negara</p>
    <ul>
         <li>Indonesia</li>
         <li>Malaysia</li>
         <li>Singapura</li>
         <li>Hongkong</li>
         <li>Vietnam</li>
         <li>India</li>
      </ul>
  </body>
</html>
