Hyperlink dalam HTML

Untuk membuat hyperlink dalam html, kita bisa menggunakan tag <a href="linktautan"> Situs Saya </a>. Bagaimana cara membuat hyperlink dalam html ? Berikut ini adalah contoh codingan untuk membuat hyperlink dalam html.

<!DOCTYPE html>
<html>
  <head>
    <title> Cara Membuat Tombol Dengan Tag Button </title>
  </head>
  <body>
    <a href="https://freeze-gans.github.io/"> Kunjungi Situs Saya </a>
  </body>
</html>
