Pengenalan HTML

HTML merupakan singkatan dari HyperText Markup Language. Html termasuk ke dalam bahasa markah yang dibangun dan disusun untuk membuat sebuah website. Jika suatu website diibaratkan seperti manusia, maka html adalah tulang dari manusia. Tetapi jika kita ingin membuat sebuah website, kita harus mencakup tiga unsur penyusun website. Apa saja penyusun nya ? Simak dibawah ini.

Tiga Unsur Penyusun Website

1. HTML
Pertama adalah html. Jika sebuah website diibaratkan seperti manusia, maka html adalah tulang dari manusia. Jika kamu ingin belajar cara membuat website, maka kamu wajib mempelajari bahasa html.

2. CSS
Html tidak hanya sendiri dalam membangun sebuah website. Tetapi html juga ditemani oleh bahasa CSS. Jika html diibaratkan seperti tulang pada manusia, maka CSS diibaratkan seperti kulit, rambut ataupun aksesoris lainnya seperti baju, celana, topi dan lainnya.

3. Javascript
Untuk membangun sebuah website, bukan hanya menggunakan html dan CSS. Tetapi kita juga membutuhkan Javascript agar website tersebut terlihat hidup. Jika html diibaratkan seperti tulang dan CSS diibaratkan seperti kulit, rambut dan sebagainya. Maka javascript tugasnya adalah menggerakkan tulang. Misalnya berjalan, berlari dan lainnya. Selain itu, dengan javascript kamu bisa membuat pop up, zoom gambar, dan lainnya.

1. Text Editor
Apa sih text editor itu ? Text Editor merupakan sebuah aplikasi atau sebuah software yang digunakan untuk menuliskan sebuah codingan. Di dalam text editor tersebut kita bebas menuliskan berbagai codingan. Mulai dari Html, Css, Javascript, Python, Php, C++, C# dan bahasa program lainnya.
