Background dalam CSS

Untuk membuat background dalam CSS, kita bisa menggunakan property background-color. Berikut ini adalah contoh codingan nya

  <!DOCTYPE Html>
  <html>
    <head>
      <title> Background dalam CSS </title>
    </head>
    <body>
      <p style="background-color: red;">Tulisan ini dengan background merah</p>
    </body>
  </html>
