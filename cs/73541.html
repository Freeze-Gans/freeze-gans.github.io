Pemformatan Teks dalam CSS

Hai semuanya, pada kali ini kita akan mempelajari tentang CSS. Materi kali ini adalah tentang Pemformatan Teks dalam CSS. Tujuan dari pemformatan teks ini adalah agar format teks menjadi apa yang kita inginkan. Contohnya kita ingin merubah warna teks nya, mengubah ukuran teks nya dan mengubah style teks nya. Bagaimana caranya ? Berikut ini adalah tutorialnya.

Cara Mengubah Warna Teks dengan CSS
Untuk mengubah warna dengan CSS, kita bisa menggunakan syntax color: warna;. Berikut ini adalah contoh codingan nya

<!DOCTYPE html>
<html>
  <head>
    <title>Pemformatan Teks dengan CSS</title>
  </head>
  <body>
    <p style="color: blue;">Teks Ini Berwarna Biru.</p>
  </body>
</html>

Hasilnya seperti dibawah ini

Cara Mengubah Letak Teks dengan CSS

Untuk mengubah letak teks dengan CSS, kita bisa menggunakan syntax text-align. Berikut ini adalah contoh codingan nya

<!DOCTYPE html>
<html>
  <head>
    <title>Pemformatan Teks dengan CSS</title>
  </head>
  <body>
    <p style="text-align: left;">Teks ini terletak pada bagian kiri.</p>
    <p style="text-align: center;">Teks ini terletak pada bagian tengah.</p>
    <p style="text-align: right;">Teks ini terletak pada bagian kanan.</p>
  </body>
</html>

Cara Membuat Teks Bergaris dengan CSS

Untuk membuat teks bergaris dengan CSS, kita bisa menggunakan syntax text-decoration. Berikut ini adalah contoh codingan nya

<!DOCTYPE html>
<html>
  <head>
    <title>Pemformatan Teks dengan CSS</title>
  </head>
  <body>
    <p style="text-decoration: line-through;">Teks ini memiliki garis ditengah.</p>
    <p style="text-decoration: overline;">Teks ini memiliki garis diatas.</p>
    <p style="text-decoration: underline;">Teks ini memiliki garis dibawah.</p>
  </body>
</html>
