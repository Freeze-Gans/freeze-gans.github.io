Operator Aritmatika dalam Javascript

Operator aritmatika merupakan operator untuk melakukan operasi aritmatika seperti penjumlahan, pengurangan, pembagian, perkalian, dan lainnya. Berikut ini adalah contoh codingan nya

<!DOCTYPE html>
<html lang="en">
  <head>
    <title>Operator Aritmatika</title>
  </head>
  <body>
    <script>
        var a = 15;
        var b = 4;
        var c = 0;
        // pengurangan
        c = a - b;
        document.write(`${a} - ${b} = ${c}<br/>`);

        // Perkalian
        c = a * b;
        document.write(`${a} * ${b} = ${c}<br/>`);

        // pemangkatan
        c = a ** b;
        document.write(`${a} ** ${b} = ${c}<br/>`);

        // Penjumlahan
        c = a + b;
        document.write(`${a} + ${b} = ${c}<br/>`);
    </script>
  </body>
</html>
