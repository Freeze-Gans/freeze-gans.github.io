Array dalam Javascript

Array merupakan struktur data yang digunakan untuk menyimpan sekumpulan data dalam satu tempat. Setiap data dalam Array memiliki indeks, sehingga kita akan mudah memprosesnya. Indeks array selalu dimulai dari angka nol (0). Contohnya seperti codingan dibawah ini

<!DOCTYPE html>
<html lang="en">
<head>
    <title>Mengambil data dari array</title>
</head>
<body>
    <script>
        var products = ["Televisi", "Radio", "Kulkas", "Lemari"];
        document.write(products[3]);
    </script>
</body>
</html>
